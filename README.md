# TTmod
A story-rich mod
Includes 14 main campaign chapters and one side campaign chapter
Overhauls the original planet Cypro
Adds new factionsnA large number of new buildings and units
We welcome everyone to try it out

Please ensure that this MOD is used alone. Mixing it with other mods may trigger the campaign's anti-mod mixing mechanisms, which could prevent the storyline from functioning properly.(Of course, these requirements can be disregarded in custom game mode.)

（The storyline is currently only available in Chinese.）
（The Mod has not yet been translated into English or other languages.）
（Supports iOS users.）
